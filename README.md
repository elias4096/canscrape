# Thesis Project - Can Scrape

The goal of this project was to develop an easy‑to‑use tool that assists with reverse engineering the CAN bus in cars.

![alt text](images/canscrape-v1.png)

# Dependencies

1. Python 3.14 or later
2. PySlide6, python-can, pyserial, numpy, pandas, scikit-learn, pyod
3. PeakCAN drivers (https://www.peak-system.com/products/hardware/external-pc-interfaces/pcan-usb/)
